# Onboarding Screenshots

Place your onboarding screenshots in this directory with the following names:

1. **login.png** - Login screen showing Google, Apple, and Email authentication options
2. **upload.png** - PDF upload interface with drag and drop area and AI labeling checkbox
3. **form-fields.png** - Form interface showing organized fields with labels and input boxes
4. **pdf-editor.png** - PDF viewer with field overlays showing editable labels
5. **field-highlight.png** - Form field with eye icon showing the highlight feature

## Image Requirements

- Format: PNG or JPG
- Recommended size: 1200x900px or similar 4:3 aspect ratio
- File size: Keep under 500KB per image for optimal loading

## How to Add Images

1. Take screenshots of your app showing each feature
2. Save them with the exact filenames listed above
3. Place them in this directory (`public/images/onboarding/`)
4. The onboarding slideshow will automatically use them
